HOW TO USE THIS WEB-APP
1. MATCHUP TAB

The Matchup tab is the home screen.

It shows the current active matchup, the timer, and the session maps.

If no matchup has been picked yet, you will see NO CURRENT MATCHUP.

Press CLICK TO GET STARTED to go to the Generator tab.

2. GENERATOR TAB

This is where you create balanced teams.

First choose a Match Maker.

Then select the players who are playing.

Press CLICK TO GENERATE MATCHUP to see team options.

Lower Difference numbers mean more balanced teams.

Difference 0 means the teams are perfectly balanced.

If the number of players is uneven, a Blitz option may appear.

This lets you only show recommended matchups for uneven teams when available.

3. PICKING A MATCHUP

When you find a matchup you like, press CLICK TO SELECT.

The web-app will save it and send you back to the Matchup tab.

The timer will start, and everyone can see the active matchup.

4. HISTORY TAB

This tab shows old matchups.

You can see the date, Match ID, Match Maker, how many times it was picked, and the Difference.

Tap a row to open it and see the full team details.

5. PLAYERS TAB

This is where you manage players and skill levels.

You can add players, remove players, and change skills.

If you edit something, the web-app will show Changes not saved.

Press SAVE to keep your changes.

6. MAP LIST TAB

This tab shows the current session maps and the full map list.

The session maps are grouped into Elimination, Blitz, and CTF.

GENERATE SESSION MAPS makes a fresh session.

SAVE SESSION PROGRESS remembers where the session should continue next time.

CLEAR SESSION MAPS removes the current session maps.

COPY SESSION MAPS copies the session maps as an image.

7. CUSTOM SESSION

BUILD CUSTOM SESSION lets you pick maps yourself.

Tap maps from the master list to build your own session.

Press SAVE CUSTOM SESSION to save it.

Press EXIT CUSTOM SESSION when you are done.

8. LOCKED / ADMIN MODE

If the web-app says LOCKED, you can still view things but you cannot change protected settings.

Tap the lock text and enter the admin password to unlock admin mode.

When finished, press LOCK to leave admin mode.

9. EASY EVERYDAY FLOW

Go to Map List and check the session maps.

Go to Generator and choose the Match Maker.

Select the players, generate matchup options, and pick the best one.

Then go back to Matchup to see the live result.

10. SIMPLE RULES TO REMEMBER

Lower Difference is better.

Always press Save after changing players.

Session maps stay visible on the Matchup tab.

History is saved automatically.

Locked mode is for viewing. Admin mode is for changing protected settings.
